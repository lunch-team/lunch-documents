## 문서 작성 양식 
```md
# 메뉴 목록 화면

## 관련 API
[/menu/getAllMenu](https://github.com/lunch-team/lunch-rest-api/wiki/Menu#모든-메뉴-조회)

## 설명
등록된 전체 메뉴를 조회한다.

## 기능
- 메뉴 리스트 조회
- 정렬 타입 선택 기능 (가나다, 방문수 순, 방문일자 순)
- 정렬 오름차순, 내림차순 토글 기능
```
